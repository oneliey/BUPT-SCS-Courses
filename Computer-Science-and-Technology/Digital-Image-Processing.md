# 数字图像处理

## 评价1

1. **开课信息**：康学净（2020秋）
2. **推荐等级**：一般推荐
3. **授课内容**：数字图像处理相关的理论，数学内容比较多
4. **考核方式**：期末开卷考试
5. **作业情况**：有 1~2 次编程作业，不定语言，一般使用 Python 或 Matlab
6. **成绩情况**：70+
7. **心得体会**：能学习到数字图像处理的数学理论知识，也能大概了解图像编辑软件处理图像的原理，不过整个课还是挺枯燥的
8. **其他方面**：由于是大四的课，所以大家也不会仔细听，但是不仔细听就听不懂，考试的时候即使是开卷也不会，不过还是能及格的
